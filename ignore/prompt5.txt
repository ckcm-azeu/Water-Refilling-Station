## PROJECT CONTEXT RECAP

Azeu Water Station is a web-based order and delivery management system for a water refilling station in the Philippines. It has 4 roles: Customer, Rider, Staff, Admin (with Super Admin). The tech stack is pure PHP (no frameworks), MySQL on XAMPP (database name: `station_azeu`), and hand-written HTML/CSS/JS. No frameworks, no Bootstrap, no Tailwind.

**Admin** is the highest operational authority. There are two admin sub-types:

1. **Admin** (`role='admin'`) — Created by Super Admin only. Has full access to everything except modifying the Super Admin account.
2. **Super Admin** (`role='super_admin'`) — Seeded into the database on system initialization. Only ONE exists. Cannot be deleted. Can modify itself and create other Admin accounts. Default credentials: username `admin`, password `admin`.

Both Admin and Super Admin share the same frontend pages inside the `admin/` folder. The PHP auth check allows both: `require_role([ROLE_ADMIN, ROLE_SUPER_ADMIN])`. Differences in permissions are enforced by the API endpoints (e.g., only super_admin can create admin accounts, only super_admin can modify itself, normal admin cannot modify super_admin).

**Admin capabilities (everything Staff can do PLUS):**
- Full inventory CRUD (add, edit, restock, remove items, upload item icons)
- Account Management with full authority: see ALL roles (customer, rider, staff, admin), create Staff and Admin accounts (Super Admin only for Admin creation), edit/delete any account except Super Admin (only Super Admin can edit itself)
- Analytics dashboard with charts (daily sales trends, order distribution, revenue, top customers, top riders, popular items)
- Session logs viewer (login/logout/failed login tracking with timestamps, roles, IPs)
- Full system settings: station name, delivery fee, max cancellations, pending expiry, low stock threshold, maintenance mode, password encryption toggle, auto-assign toggle, timezone, force dark mode, UI color customization (primary/secondary/accent/surface colors via color pickers), max login attempts, default item list management
- Export analytics reports as downloadable documents

**Admin CANNOT:**
- Place orders (only customers can)
- Set rider availability (only riders can)
- Register through the public registration form (must be created by Super Admin)

## ADMIN PERMISSION DETAILS

### Account Management Scope

| Logged-in As | Can See | Can Create | Can Edit | Can Delete |
|---|---|---|---|---|
| Admin | Customer, Rider, Staff | Customer, Rider, Staff | Customer, Rider, Staff (NOT super_admin) | Customer, Rider, Staff (NOT super_admin) |
| Super Admin | Customer, Rider, Staff, Admin | Customer, Rider, Staff, Admin | ALL including self | ALL except self (super_admin cannot be deleted) |

### System Settings Access

Only Admin and Super Admin can access system settings. All setting changes apply system-wide.

**Critical setting behaviors:**

1. **Maintenance Mode** — When ON, all non-admin users see `maintenance.php` on login. Current sessions of lower roles still function until they navigate/refresh. Admins see a warning banner on their dashboard but can operate normally.

2. **Encrypt DB Passwords** — Toggle triggers a migration:
   - Turning ON: All plaintext passwords in `users` table are encrypted using AESCrypt
   - Turning OFF: All encrypted passwords are decrypted back to plaintext
   - Show a confirmation warning before toggling
   - Log every password migration action

3. **Auto Assign Every Order** — When ON, after Staff/Admin confirms an order, the system automatically runs the auto-assign rider algorithm. If no rider available, order stays at Confirmed with auto-generated staff comment "Auto-assign: No riders available."

4. **Force Dark Mode** — When ON, all users see dark mode regardless of individual preference. The theme toggle button is disabled/hidden for all users.

5. **UI Colors** — Primary, secondary, accent, and surface colors can be customized via color pickers. Changes apply system-wide immediately via CSS custom properties injected by PHP on the `<html>` element's `style` attribute.

### `admin/js/inventory.js`

```javascript
document.addEventListener('DOMContentLoaded', function() {

  let currentFilter = 'all';
  let currentPage = 1;
  let searchText = '';
  const perPage = 50;
  const lowStockThreshold = parseInt('<?= get_setting("low_stock_threshold") ?>') || 10;

  loadInventory();

  // Filters
  document.querySelectorAll('#inventoryFilters .filter-btn').forEach(btn => {
    btn.addEventListener('click', function() {
      document.querySelectorAll('#inventoryFilters .filter-btn').forEach(b => b.classList.remove('active'));
      this.classList.add('active');
      currentFilter = this.dataset.filter;
      currentPage = 1;
      loadInventory();
    });
  });

  // Search
  let searchTimeout;
  document.getElementById('inventorySearch').addEventListener('input', function() {
    clearTimeout(searchTimeout);
    searchTimeout = setTimeout(() => { searchText = this.value.trim(); currentPage = 1; loadInventory(); }, 300);
  });

  function loadInventory() {
    const params = {
      status_filter: ['active','inactive','out_of_stock'].includes(currentFilter) ? currentFilter : '',
      low_stock: currentFilter === 'low_stock' ? '1' : '',
      search: searchText, page: currentPage
    };
    document.getElementById('inventoryBody').innerHTML = '<tr class="loading-row"><td colspan="7"><div class="spinner"></div></td></tr>';
    document.getElementById('emptyInventory').style.display = 'none';

    ajaxGet('/api/inventory/get_items.php', params).then(data => {
      if (!data.items || data.items.length === 0) {
        document.getElementById('inventoryBody').innerHTML = '';
        document.getElementById('emptyInventory').style.display = 'flex';
        document.getElementById('inventoryPagination').innerHTML = '';
        return;
      }
      document.getElementById('emptyInventory').style.display = 'none';
      renderInventory(data.items);
      renderPagination(data.total, data.page, data.per_page);
    }).catch(() => showToast('Failed to load inventory', 'error'));
  }

  function renderInventory(items) {
    const tbody = document.getElementById('inventoryBody');
    tbody.innerHTML = '';
    items.forEach(item => {
      const isLow = parseInt(item.stock_count) <= lowStockThreshold;
      const isZero = parseInt(item.stock_count) === 0;
      const tr = document.createElement('tr');
      if (isLow) tr.classList.add('low-stock-row');

      const statusLabels = { active: 'Active', inactive: 'Inactive', out_of_stock: 'Out of Stock' };

      tr.innerHTML = `
        <td class="item-cell">
          <img src="${item.item_icon || '/assets/images/default-item.png'}" class="item-icon" alt="">
          <span class="item-name">${escapeHtml(item.item_name)}</span>
        </td>
        <td class="${isLow ? 'text-danger' : ''}" style="font-weight:700;">
          ${isLow ? '<span class="material-icons" style="font-size:16px;vertical-align:middle;color:var(--warning);">warning</span> ' : ''}
          ${item.stock_count}
        </td>
        <td style="color:var(--primary);font-weight:600;">₱${parseFloat(item.price).toFixed(2)}</td>
        <td>
          ${isLow
            ? '<span style="color:var(--warning);"><span class="material-icons" style="font-size:16px;vertical-align:middle;">warning</span> Low Stock</span>'
            : '<span style="color:var(--success);"><span class="material-icons" style="font-size:16px;vertical-align:middle;">check_circle</span> OK</span>'}
        </td>
        <td>${item.last_restocked_at ? formatDate(item.last_restocked_at) : '<span class="text-muted">Never</span>'}</td>
        <td><span class="badge badge-${item.status === 'out_of_stock' ? 'cancelled' : item.status === 'active' ? 'confirmed' : 'pending'}">${statusLabels[item.status]}</span></td>
        <td class="actions-cell">
          <button class="btn-icon btn-info" title="Edit" onclick="editItem(${item.id})"><span class="material-icons">edit</span></button>
          <button class="btn-icon btn-success" title="Restock" onclick="restockItem(${item.id}, '${escapeHtml(item.item_name)}')"><span class="material-icons">add_box</span></button>
          <button class="btn-icon btn-danger" title="Remove" onclick="removeItem(${item.id}, '${escapeHtml(item.item_name)}')"><span class="material-icons">delete</span></button>
        </td>
      `;
      tbody.appendChild(tr);
    });
  }

  // ADD ITEM
  document.getElementById('addItemBtn').addEventListener('click', function() {
    // Fetch default items for dropdown
    ajaxGet('/api/settings/get_settings.php', { scope: 'default_items' }).then(data => {
      const defaultItems = data.default_items || [];
      const optionsHtml = defaultItems.map(d => `<option value="${escapeHtml(d.item_name)}">${escapeHtml(d.item_name)}</option>`).join('');

      Swal.fire({
        title: 'Add New Item',
        html: `
          <div style="text-align:left;">
            <label style="font-weight:500;display:block;margin-bottom:4px;">Item Name</label>
            <select id="addItemSelect" class="swal2-select" style="width:100%;margin-bottom:8px;">
              <option value="">— Select or type custom —</option>
              ${optionsHtml}
              <option value="__custom__">Custom name...</option>
            </select>
            <input type="text" id="addItemCustomName" class="swal2-input" placeholder="Custom item name" style="display:none;">
            <input type="number" id="addItemStock" class="swal2-input" placeholder="Stock Count" min="0" step="1">
            <input type="number" id="addItemPrice" class="swal2-input" placeholder="Price (₱)" min="0" step="0.01">
            <label style="font-weight:500;display:block;margin-top:12px;margin-bottom:4px;">Item Icon (optional)</label>
            <input type="file" id="addItemIcon" class="swal2-file" accept="image/*">
          </div>
        `,
        showCancelButton: true,
        confirmButtonText: 'Add Item', confirmButtonColor: 'var(--primary)',
        didOpen: () => {
          document.getElementById('addItemSelect').addEventListener('change', function() {
            document.getElementById('addItemCustomName').style.display = this.value === '__custom__' ? 'block' : 'none';
          });
        },
        preConfirm: () => {
          const select = document.getElementById('addItemSelect').value;
          const custom = document.getElementById('addItemCustomName').value.trim();
          const itemName = select === '__custom__' ? custom : select;
          const stock = document.getElementById('addItemStock').value;
          const price = document.getElementById('addItemPrice').value;
          const iconFile = document.getElementById('addItemIcon').files[0];

          if (!itemName) { Swal.showValidationMessage('Item name is required'); return false; }
          if (!stock || parseInt(stock) < 0) { Swal.showValidationMessage('Valid stock count required'); return false; }
          if (!price || parseFloat(price) < 0) { Swal.showValidationMessage('Valid price required'); return false; }

          return { item_name: itemName, stock_count: stock, price: price, icon: iconFile };
        }
      }).then(result => {
        if (result.isConfirmed) {
          const formData = new FormData();
          formData.append('item_name', result.value.item_name);
          formData.append('stock_count', result.value.stock_count);
          formData.append('price', result.value.price);
          formData.append('csrf_token', getCsrfToken());
          if (result.value.icon) formData.append('item_icon', result.value.icon);

          fetch('/api/inventory/add_item.php', { method: 'POST', body: formData })
            .then(r => r.json())
            .then(d => { if(d.success) { showToast('Item added!','success'); loadInventory(); } else showToast(d.message||'Failed','error'); })
            .catch(() => showToast('Error','error'));
        }
      });
    });
  });

  // EDIT ITEM
  window.editItem = function(itemId) {
    ajaxGet('/api/inventory/get_items.php', { item_id: itemId }).then(data => {
      const item = data.items?.find(i => i.id === itemId) || data.item;
      if (!item) { showToast('Item not found','error'); return; }

      Swal.fire({
        title: 'Edit Item',
        html: `
          <div style="text-align:left;">
            <input type="text" id="editItemName" class="swal2-input" value="${escapeHtml(item.item_name)}" placeholder="Item Name">
            <input type="number" id="editItemPrice" class="swal2-input" value="${item.price}" placeholder="Price" step="0.01">
            <label style="font-weight:500;display:block;margin-top:12px;">Status</label>
            <select id="editItemStatus" class="swal2-select" style="width:100%;">
              <option value="active" ${item.status==='active'?'selected':''}>Active</option>
              <option value="inactive" ${item.status==='inactive'?'selected':''}>Inactive</option>
              <option value="out_of_stock" ${item.status==='out_of_stock'?'selected':''}>Out of Stock</option>
            </select>
            <label style="font-weight:500;display:block;margin-top:12px;">Replace Icon (optional)</label>
            <input type="file" id="editItemIcon" class="swal2-file" accept="image/*">
          </div>
        `,
        showCancelButton: true,
        confirmButtonText: 'Save Changes', confirmButtonColor: 'var(--primary)',
        preConfirm: () => {
          const name = document.getElementById('editItemName').value.trim();
          const price = document.getElementById('editItemPrice').value;
          if (!name) { Swal.showValidationMessage('Name required'); return false; }
          if (!price || parseFloat(price)<0) { Swal.showValidationMessage('Valid price required'); return false; }
          return {
            item_name: name, price: price,
            status: document.getElementById('editItemStatus').value,
            icon: document.getElementById('editItemIcon').files[0]
          };
        }
      }).then(result => {
        if (result.isConfirmed) {
          const formData = new FormData();
          formData.append('item_id', itemId);
          formData.append('item_name', result.value.item_name);
          formData.append('price', result.value.price);
          formData.append('status', result.value.status);
          formData.append('csrf_token', getCsrfToken());
          if (result.value.icon) formData.append('item_icon', result.value.icon);

          fetch('/api/inventory/update_item.php', { method: 'POST', body: formData })
            .then(r => r.json())
            .then(d => { if(d.success) { showToast('Updated!','success'); loadInventory(); } else showToast(d.message||'Failed','error'); })
            .catch(() => showToast('Error','error'));
        }
      });
    });
  };

  // RESTOCK
  window.restockItem = function(itemId, itemName) {
    Swal.fire({
      title: 'Restock: ' + itemName,
      html: '<input type="number" id="restockQty" class="swal2-input" placeholder="Quantity to add" min="1">',
      showCancelButton: true, confirmButtonText: 'Restock', confirmButtonColor: 'var(--success)',
      preConfirm: () => {
        const qty = parseInt(document.getElementById('restockQty').value);
        if (!qty || qty < 1) { Swal.showValidationMessage('Enter valid quantity'); return false; }
        return qty;
      }
    }).then(r => {
      if (r.isConfirmed) {
        ajaxPost('/api/inventory/restock.php', { item_id: itemId, add_quantity: r.value })
          .then(d => { if(d.success) { showToast('Restocked!','success'); loadInventory(); } else showToast(d.message||'Failed','error'); })
          .catch(() => showToast('Error','error'));
      }
    });
  };

  // REMOVE
  window.removeItem = function(itemId, itemName) {
    Swal.fire({
      title: 'Remove Item?',
      text: `Are you sure you want to delete "${itemName}"?`,
      icon: 'warning', showCancelButton: true,
      confirmButtonText: 'Yes, Remove', confirmButtonColor: '#EF5350'
    }).then(r => {
      if (r.isConfirmed) {
        ajaxPost('/api/inventory/delete_item.php', { item_id: itemId })
          .then(d => { if(d.success) { showToast('Removed','success'); loadInventory(); } else showToast(d.message||'Failed','error'); })
          .catch(() => showToast('Error','error'));
      }
    });
  };

  // HELPERS
  function escapeHtml(t) { const d=document.createElement('div'); d.textContent=t||''; return d.innerHTML; }
  function formatDate(d) { if(!d) return '—'; return new Date(d).toLocaleDateString('en-US',{month:'short',day:'numeric',year:'numeric'}); }

  function renderPagination(total, page, perPage) {
    const totalPages = Math.ceil(total / perPage);
    const c = document.getElementById('inventoryPagination'); c.innerHTML = '';
    if (totalPages <= 1) return;
    const prev = document.createElement('button'); prev.className='page-btn'; prev.textContent='Previous'; prev.disabled=page<=1;
    prev.addEventListener('click',()=>{currentPage--;loadInventory();}); c.appendChild(prev);
    for(let i=Math.max(1,page-2);i<=Math.min(totalPages,page+2);i++){
      const pb=document.createElement('button');pb.className=`page-btn ${i===page?'active':''}`;pb.textContent=i;
      pb.addEventListener('click',()=>{currentPage=i;loadInventory();});c.appendChild(pb);
    }
    const next=document.createElement('button');next.className='page-btn';next.textContent='Next';next.disabled=page>=totalPages;
    next.addEventListener('click',()=>{currentPage++;loadInventory();});c.appendChild(next);
    const info=document.createElement('span');info.className='pagination-info';
    info.textContent=`Showing ${(page-1)*perPage+1}–${Math.min(page*perPage,total)} of ${total}`;c.appendChild(info);
  }

});
```

```javascript
document.addEventListener('DOMContentLoaded', function() {

  function loadAnalytics() {
    const params = { period: currentPeriod, date_from: dateFrom, date_to: dateTo };

    // Fetch sales data

          label: 'Revenue (₱)',
          data: data.daily_revenue || [],
          borderColor: getComputedStyle(document.documentElement).getPropertyValue('--primary').trim(),

      });

      // Orders Per Day (Bar)
      renderChart('ordersPerDayChart', 'bar', {

      });

      // Top Items (Horizontal Bar)
      renderChart('topItemsChart', 'bar', {
        labels: (data.top_items || []).map(i => i.item_name),
        datasets: [{
          label: 'Qty Sold',
          data: (data.top_items || []).map(i => i.total_qty),
          backgroundColor: '#7E57C2', borderRadius: 6
        }]
      }, { indexAxis: 'y' });

      // Monthly Comparison (Bar)
      renderChart('monthlyCompChart', 'bar', {
        labels: ['Orders', 'Revenue (₱)', 'Cancellations'],
        datasets: [
          { label: 'Last Month', data: [data.last_month_orders||0, data.last_month_revenue||0, data.last_month_cancellations||0], backgroundColor: '#90CAF9', borderRadius: 6 },

      ));
    }).catch(() => showToast('Failed to load analytics','error'));

    // Order Status Distribution (Doughnut)
    ajaxGet('/api/analytics/get_order_stats.php', params).then(data => {

      renderChart('statusDistChart', 'doughnut', {

  function renderChart(canvasId, type, data, extraOptions = {}) {
    if (charts[canvasId]) charts[canvasId].destroy();
    const ctx = document.getElementById(canvasId).getContext('2d');
    const isDark = document.documentElement.getAttribute('data-theme') === 'dark';
    charts[canvasId] = new Chart(ctx, {
      type: type,
      data: data,
      options: {
        responsive: true,
        maintainAspectRatio: true,
        plugins: {
          legend: { labels: { color: isDark ? '#E8E8F0' : '#1A1A2E' } }

          x: { ticks: { color: isDark ? '#A8A8C0' : '#4A4A6A' }, grid: { color: isDark ? '#2A2A40' : '#E0E4EC' } },

  function getStatusColor(status) {

  function renderTableRows(tbodyId, rowsHtml) {

  });

  function formatStatus(s) {
    const l = { pending:'Pending', confirmed:'Confirmed', assigned:'Assigned', on_delivery:'On Delivery', delivered:'Delivered', accepted:'Accepted', ready_for_pickup:'Ready for Pickup', picked_up:'Picked Up', cancelled:'Cancelled' };

  }
  function formatDate(d) { if(!d) return '—'; return new Date(d).toLocaleDateString('en-US',{month:'short',day:'numeric',year:'numeric'}); }
```

### `admin/js/session_logs.js`

```javascript
document.addEventListener('DOMContentLoaded', function() {

  let currentFilter = 'all';
  let currentPage = 1;
  let searchText = '';
  let roleFilter = '';
  let dateFrom = '';
  let dateTo = '';
  const perPage = 50;

  loadLogs();

  // Filters
  document.querySelectorAll('#logFilters .filter-btn').forEach(btn => {
    btn.addEventListener('click', function() {
      document.querySelectorAll('#logFilters .filter-btn').forEach(b => b.classList.remove('active'));
      this.classList.add('active');
      currentFilter = this.dataset.filter;
      currentPage = 1;
      loadLogs();
    });
  });

  // Search
  let searchTimeout;
  document.getElementById('logSearch').addEventListener('input', function() {
    clearTimeout(searchTimeout);
    searchTimeout = setTimeout(() => { searchText = this.value.trim(); currentPage = 1; loadLogs(); }, 300);
  });

  // Role filter
  document.getElementById('roleFilter').addEventListener('change', function() {
    roleFilter = this.value; currentPage = 1; loadLogs();
  });

  // Date range
  document.getElementById('logDateFrom').addEventListener('change', function() { dateFrom = this.value; currentPage = 1; loadLogs(); });
  document.getElementById('logDateTo').addEventListener('change', function() { dateTo = this.value; currentPage = 1; loadLogs(); });

  function loadLogs() {
    // Session logs are fetched from the session_logs table
    // The API endpoint would be something like api/analytics/get_session_logs.php
    // or a dedicated endpoint — for simplicity, we can fetch from a generic endpoint
    const params = {
      action_filter: currentFilter === 'all' ? '' : currentFilter,
      role_filter: roleFilter,
      search: searchText,
      date_from: dateFrom,
      date_to: dateTo,
      page: currentPage
    };

    document.getElementById('logsBody').innerHTML = '<tr class="loading-row"><td colspan="6"><div class="spinner"></div></td></tr>';
    document.getElementById('emptyLogs').style.display = 'none';

    // Since there's no dedicated session_logs API endpoint in the original API list,
    // this data can also be loaded via PHP server-side rendering or via a new endpoint.
    // For consistency, use AJAX to an endpoint that returns session_logs data.
    // Assume: GET api/analytics/get_session_logs.php exists or session logs are handled via get_dashboard_stats
    ajaxGet('/api/settings/get_settings.php', { scope: 'session_logs', ...params })
      .then(data => {
        if (!data.logs || data.logs.length === 0) {
          document.getElementById('logsBody').innerHTML = '';
          document.getElementById('emptyLogs').style.display = 'flex';
          document.getElementById('logsPagination').innerHTML = '';
          return;
        }
        document.getElementById('emptyLogs').style.display = 'none';
        renderLogs(data.logs, data.page, data.per_page);
        renderPagination(data.total, data.page, data.per_page);
      })
      .catch(() => showToast('Failed to load session logs', 'error'));
  }

  function renderLogs(logs, page, perPage) {
    const tbody = document.getElementById('logsBody');
    tbody.innerHTML = '';

    const actionLabels = { login: 'Login', logout: 'Logout', failed_login: 'Failed Login' };
    const actionBadges = { login: 'badge-confirmed', logout: 'badge-pending', failed_login: 'badge-cancelled' };
    const actionIcons = { login: 'login', logout: 'logout', failed_login: 'error' };

    logs.forEach((log, index) => {
      const tr = document.createElement('tr');
      if (log.action === 'failed_login') tr.classList.add('failed-row');

      const rowNum = (page - 1) * perPage + index + 1;
      const roleLabel = log.role.replace('_', ' ').replace(/\b\w/g, l => l.toUpperCase());

      tr.innerHTML = `
        <td>${rowNum}</td>
        <td>${formatDate(log.created_at)}</td>
        <td><span class="badge badge-${log.role.replace('_','-')}">${roleLabel}</span></td>
        <td><strong>${escapeHtml(log.username)}</strong></td>
        <td>
          <span class="badge ${actionBadges[log.action]}">
            <span class="material-icons" style="font-size:14px;vertical-align:middle;margin-right:4px;">${actionIcons[log.action]}</span>
            ${actionLabels[log.action]}
          </span>
        </td>
        <td><code>${escapeHtml(log.ip_address)}</code></td>
      `;
      tbody.appendChild(tr);
    });
  }

  function escapeHtml(t) { const d=document.createElement('div'); d.textContent=t||''; return d.innerHTML; }
  function formatDate(d) {
    if(!d) return '—';
    return new Date(d).toLocaleDateString('en-US',{month:'short',day:'numeric',year:'numeric'})
      + ' ' + new Date(d).toLocaleTimeString('en-US',{hour:'2-digit',minute:'2-digit',second:'2-digit'});
  }

  function renderPagination(total, page, perPage) {
    const totalPages = Math.ceil(total / perPage);
    const c = document.getElementById('logsPagination'); c.innerHTML = '';
    if (totalPages <= 1) return;
    const prev = document.createElement('button'); prev.className='page-btn'; prev.textContent='Previous'; prev.disabled=page<=1;
    prev.addEventListener('click',()=>{currentPage--;loadLogs();}); c.appendChild(prev);
    for(let i=Math.max(1,page-2);i<=Math.min(totalPages,page+2);i++){
      const pb=document.createElement('button');pb.className=`page-btn ${i===page?'active':''}`;pb.textContent=i;
      pb.addEventListener('click',()=>{currentPage=i;loadLogs();});c.appendChild(pb);
    }
    const next=document.createElement('button');next.className='page-btn';next.textContent='Next';next.disabled=page>=totalPages;
    next.addEventListener('click',()=>{currentPage++;loadLogs();});c.appendChild(next);
    const info=document.createElement('span');info.className='pagination-info';
    info.textContent=`Showing ${(page-1)*perPage+1}–${Math.min(page*perPage,total)} of ${total}`;c.appendChild(info);
  }

});
```

## IMPORTANT IMPLEMENTATION NOTES FOR ADMIN PAGES

1. **`require_role([ROLE_ADMIN, ROLE_SUPER_ADMIN])`** — Both admin and super_admin can access all admin pages. The API endpoints handle the permission differences (e.g., only super_admin can create admin accounts).

2. **Pages duplicated from Staff** (`manage_orders`, `order_history`, `rider_statistics`): These are functionally identical to the staff versions. Duplicate the code entirely into the admin folder. The only difference is the `require_role()` call at the top. This follows the project requirement of duplicating code per role folder for easier independent maintenance.

3. **Account Management differences from Staff version:**
   - Admin sees ALL roles: customer, rider, staff, admin (not super_admin unless the viewer IS super_admin)
   - Admin can create staff accounts. Super Admin can also create admin accounts.
   - Inject a PHP variable into JS: `const isSuperAdmin = <?= json_encode($_SESSION['role'] === ROLE_SUPER_ADMIN) ?>;`
   - Use this variable to show/hide the "Admin" option in role dropdowns
   - Super Admin row in the table (if current user is super_admin viewing accounts) should show a lock icon and no delete button

4. **Inventory page** is the FULL CRUD version:
   - Add item (with optional icon upload, default item name dropdown from `default_items` table)
   - Edit item (name, price, status, replace icon)
   - Restock (add quantity, update last_restocked_at)
   - Remove item (with protection against items in active orders)
   - File uploads use `FormData` and `fetch()` instead of `ajaxPost()` because file uploads need `multipart/form-data`
   - CSRF token is included in the FormData manually
