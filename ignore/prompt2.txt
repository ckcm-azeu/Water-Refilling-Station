## PROJECT CONTEXT RECAP

Azeu Water Station is a web-based order and delivery management system for a water refilling station in the Philippines. It has 4 roles: Customer, Rider, Staff, Admin (with Super Admin). The tech stack is pure PHP (no frameworks), MySQL on XAMPP, and hand-written HTML/CSS/JS. Database name is `station_azeu`.

**Customers** are the end users who:
- Register themselves (account starts as `pending`, must be approved by Staff/Admin)
- Place orders for water and station products
- Track order status from placement through delivery
- Receive deliveries from assigned riders or pick up at store
- Download receipts
- Manage their profile, addresses, and settings
- Can reorder from order history
- Can submit cancellation appeals if their account gets flagged

## ORDER LIFECYCLE (Customer Perspective)

### Delivery Orders (Cash on Delivery)
```
Customer places order → PENDING
Staff/Admin confirms → CONFIRMED (stock decremented here)
Staff/Admin assigns rider → ASSIGNED
Rider marks on the way → ON DELIVERY (expected_delivery_date set to today)
Rider marks arrived → DELIVERED (income counted, delivered_at set)
Customer optionally clicks Accept → ACCEPTED
```

### Pickup Orders
```
Customer places order → PENDING
Staff/Admin confirms → CONFIRMED (stock decremented)
Staff/Admin marks ready → READY FOR PICKUP
Staff marks collected → PICKED UP (income counted)
Customer optionally clicks Accept → ACCEPTED
```

### Customer Cancellation Rules
- Customer can ONLY cancel orders at `PENDING` status
- A cancellation reason is REQUIRED (free text)
- Each cancellation increments `cancellation_count`
- If `cancellation_count` reaches `max_cancellation` setting (default 5), account status becomes `flagged`
- Flagged customers CANNOT place new orders
- Flagged customers can submit an appeal (once at a time) for Staff/Admin to review
- `cancellation_count` resets to 0 on the first day of each month (but flagged status persists until appeal approved)

### Customer "Accept/Received" Action
- Only available when order status is `DELIVERED` or `PICKED UP`
- Optional — customer doesn't have to click it
- Sets `customer_confirmed = 1`, `customer_confirmed_at = NOW()`, status to `ACCEPTED`
- This is an acknowledgment, does NOT affect income counting

### 1. `customer/dashboard.php`

**PHP Logic (top of file):**
```
1. Include auth_check.php, require_role([ROLE_CUSTOMER])
2. Get current user data from session
3. Check account status:
   - If status='pending' → set $is_pending = true
   - If status='flagged' → set $is_flagged = true
   - If status='active' → set $is_active = true
4. If active:
   - Fetch dashboard stats via PHP (or page loads and JS fetches via AJAX):
     - Total orders count: SELECT COUNT(*) FROM orders WHERE customer_id = user_id
     - Pending orders count: SELECT COUNT(*) FROM orders WHERE customer_id = user_id AND status = 'pending'
     - Delivered orders count: SELECT COUNT(*) FROM orders WHERE customer_id = user_id AND status IN ('delivered','accepted','picked_up')
   - Fetch pending orders preview (max 3):
     SELECT o.*, GROUP_CONCAT(oi.item_name) as item_names, GROUP_CONCAT(oi.item_icon) as item_icons,
     u.full_name as rider_name
     FROM orders o
     LEFT JOIN order_items oi ON o.id = oi.order_id
     LEFT JOIN users u ON o.rider_id = u.id
     WHERE o.customer_id = user_id AND o.status IN ('pending','confirmed','assigned','on_delivery')
     GROUP BY o.id
     ORDER BY o.created_at DESC
     LIMIT 3
5. Include header.php, sidebar.php
```

### `customer/js/dashboard.js`

```javascript
document.addEventListener('DOMContentLoaded', function() {

  // If account is flagged, handle appeal submission
  const appealBtn = document.getElementById('submitAppealBtn');
  if (appealBtn) {
    appealBtn.addEventListener('click', function() {
      // Check if there's already a pending appeal
      ajaxGet('/api/appeals/get.php', {status_filter: 'pending'})
        .then(data => {
          if (data.appeals && data.appeals.length > 0) {
            Swal.fire({
              icon: 'info',
              title: 'Appeal Already Pending',
              text: 'You already have a pending appeal. Please wait for staff to review it.',
              confirmButtonColor: 'var(--primary)'
            });
          } else {
            // Show appeal form
            Swal.fire({
              title: 'Submit Cancellation Appeal',
              html: `
                <p style="margin-bottom: 15px; color: var(--text-secondary);">
                  Explain why your cancellations were necessary and request to have your account unflagged.
                </p>
                <textarea id="appealReason" class="swal2-textarea"
                  placeholder="Enter your reason for appeal..." rows="5"
                  style="resize: vertical;"></textarea>
              `,
              showCancelButton: true,
              confirmButtonText: 'Submit Appeal',
              cancelButtonText: 'Cancel',
              confirmButtonColor: 'var(--warning)',
              preConfirm: () => {
                const reason = document.getElementById('appealReason').value.trim();
                if (!reason) {
                  Swal.showValidationMessage('Please enter a reason for your appeal');
                  return false;
                }
                return reason;
              }
            }).then(result => {
              if (result.isConfirmed) {
                ajaxPost('/api/appeals/submit.php', { reason: result.value })
                  .then(data => {
                    if (data.success) {
                      Swal.fire({
                        icon: 'success',
                        title: 'Appeal Submitted',
                        text: 'Your appeal has been submitted for review.',
                        confirmButtonColor: 'var(--primary)'
                      }).then(() => location.reload());
                    } else {
                      showToast(data.message || 'Failed to submit appeal', 'error');
                    }
                  });
              }
            });
          }
        });
    });
  }

  // Dashboard stats could be loaded via AJAX if preferred
  // For now, they're rendered server-side by PHP

});
```

```javascript
document.addEventListener('DOMContentLoaded', function() {

  // State
  let allItems = [];        // All inventory items loaded
  let selectedItems = {};   // { inventory_id: { ...itemData, quantity: X } }
  let currentPage = 1;
  const itemsPerPage = 20;  // ORDER_ITEMS_PER_PAGE
  const deliveryFee = parseFloat('{PHP_DELIVERY_FEE}'); // Injected by PHP
  let reorderItems = null;  // Pre-selected items from reorder (injected by PHP if applicable)

  // Load items on page load
  loadItems();

  // Check for reorder data (PHP can output a JS variable)
  // e.g., const reorderData = <?php echo json_encode($reorder_items ?? null); ?>;
  // If reorderData exists, pre-select those items after loading

  function loadItems(search = '') {
    showLoading();
    const params = { status_filter: 'active', page: currentPage, search: search };
    ajaxGet('/api/inventory/get_items.php', params)
      .then(data => {
        allItems = data.items;
        renderItems(data.items);
        renderPagination(data.total, data.page, data.per_page);
        hideLoading();

        // If reorder, pre-select matching items
        if (reorderData && currentPage === 1) {
          preselectReorderItems(reorderData);
          reorderData = null; // Only do this once
        }
      })
      .catch(err => {
        showToast('Failed to load items', 'error');
        hideLoading();
      });
  }

  function renderItems(items) {
    const grid = document.getElementById('itemsGrid');
    // Clear existing items (keep spinner)
    grid.querySelectorAll('.item-card').forEach(c => c.remove());

    items.forEach(item => {
      const isSelected = selectedItems[item.id] !== undefined;
      const isOutOfStock = item.stock_count <= 0 || item.status === 'out_of_stock';
      const quantity = isSelected ? selectedItems[item.id].quantity : 1;

      const card = document.createElement('div');
      card.className = `item-card ${isSelected ? 'selected' : ''} ${isOutOfStock ? 'out-of-stock' : ''}`;
      card.dataset.itemId = item.id;
      card.innerHTML = `
        <div class="item-card-check"><span class="material-icons">check_circle</span></div>
        ${isOutOfStock ? '<div class="out-of-stock-overlay">Out of Stock</div>' : ''}
        <img src="${item.item_icon || '/assets/images/default-item.png'}" class="item-card-icon" alt="${item.item_name}">
        <div class="item-card-name">${item.item_name}</div>
        <div class="item-card-price">₱${parseFloat(item.price).toFixed(2)}</div>
        <div class="item-card-stock ${item.stock_count <= 10 ? 'low-stock' : ''}">
          ${item.stock_count} in stock
        </div>
        <div class="qty-control">
          <button class="qty-btn qty-minus" data-item-id="${item.id}" ${quantity <= 1 ? 'disabled' : ''}>
            <span class="material-icons">remove</span>
          </button>
          <input type="number" class="qty-input" data-item-id="${item.id}"
                 value="${quantity}" min="1" max="${item.stock_count}" step="1">
          <button class="qty-btn qty-plus" data-item-id="${item.id}" ${quantity >= item.stock_count ? 'disabled' : ''}>
            <span class="material-icons">add</span>
          </button>
        </div>
      `;

      // Click card to select/deselect (not on qty controls)
      card.addEventListener('click', function(e) {
        if (e.target.closest('.qty-control')) return; // Don't toggle on qty click
        if (isOutOfStock) return;
        toggleItemSelection(item, card);
      });

      grid.appendChild(card);
    });

    // Attach qty control event listeners
    attachQtyListeners();
  }

  function toggleItemSelection(item, cardElement) {
    if (selectedItems[item.id]) {
      // Deselect
      delete selectedItems[item.id];
      cardElement.classList.remove('selected');
    } else {
      // Select
      const qtyInput = cardElement.querySelector('.qty-input');
      const quantity = parseInt(qtyInput.value) || 1;
      selectedItems[item.id] = {
        inventory_id: item.id,
        item_name: item.item_name,
        item_icon: item.item_icon,
        price: parseFloat(item.price),
        stock_count: item.stock_count,
        quantity: quantity
      };
      cardElement.classList.add('selected');
    }
    updateCheckoutBar();
  }

  function attachQtyListeners() {
    // Minus buttons
    document.querySelectorAll('.qty-minus').forEach(btn => {
      btn.addEventListener('click', function(e) {
        e.stopPropagation();
        const itemId = this.dataset.itemId;
        const input = document.querySelector(`.qty-input[data-item-id="${itemId}"]`);
        let val = parseInt(input.value) - 1;
        if (val < 1) val = 1;
        input.value = val;
        updateSelectedQuantity(itemId, val);
      });
    });

    // Plus buttons
    document.querySelectorAll('.qty-plus').forEach(btn => {
      btn.addEventListener('click', function(e) {
        e.stopPropagation();
        const itemId = this.dataset.itemId;
        const input = document.querySelector(`.qty-input[data-item-id="${itemId}"]`);
        const max = parseInt(input.max);
        let val = parseInt(input.value) + 1;
        if (val > max) val = max;
        input.value = val;
        updateSelectedQuantity(itemId, val);
      });
    });

    // Direct input change
    document.querySelectorAll('.qty-input').forEach(input => {
      input.addEventListener('change', function(e) {
        e.stopPropagation();
        const itemId = this.dataset.itemId;
        let val = parseInt(this.value);
        const max = parseInt(this.max);
        if (isNaN(val) || val < 1) val = 1;
        if (val > max) val = max;
        this.value = val;
        updateSelectedQuantity(itemId, val);
      });

      // Prevent card click when clicking input
      input.addEventListener('click', function(e) { e.stopPropagation(); });
    });
  }

  function updateSelectedQuantity(itemId, quantity) {
    if (selectedItems[itemId]) {
      selectedItems[itemId].quantity = quantity;
      updateCheckoutBar();
    }
  }

  function updateCheckoutBar() {
    const count = Object.keys(selectedItems).length;
    const bar = document.getElementById('checkoutBar');
    document.getElementById('selectedCount').textContent = count;
    bar.style.display = count > 0 ? 'flex' : 'none';
  }

  // Search
  let searchTimeout;
  document.getElementById('itemSearch').addEventListener('input', function() {
    clearTimeout(searchTimeout);
    searchTimeout = setTimeout(() => {
      currentPage = 1;
      loadItems(this.value.trim());
    }, 300);
  });

  // Pagination functions (render page buttons, handle clicks)
  function renderPagination(total, page, perPage) {
    const totalPages = Math.ceil(total / perPage);
    const container = document.getElementById('itemsPagination');
    container.innerHTML = '';

    if (totalPages <= 1) return;

    // Previous button
    const prevBtn = createPageBtn('Previous', page > 1, () => { currentPage--; loadItems(); });
    container.appendChild(prevBtn);

    // Page numbers (show max 5 around current)
    for (let i = Math.max(1, page-2); i <= Math.min(totalPages, page+2); i++) {
      const pageBtn = createPageBtn(i, true, () => { currentPage = i; loadItems(); });
      if (i === page) pageBtn.classList.add('active');
      container.appendChild(pageBtn);
    }

    // Next button
    const nextBtn = createPageBtn('Next', page < totalPages, () => { currentPage++; loadItems(); });
    container.appendChild(nextBtn);

    // Items count
    const info = document.createElement('span');
    info.className = 'pagination-info';
    info.textContent = `Showing ${(page-1)*perPage + 1}–${Math.min(page*perPage, total)} of ${total} items`;
    container.appendChild(info);
  }

  function createPageBtn(text, enabled, onClick) {
    const btn = document.createElement('button');
    btn.className = 'page-btn';
    btn.textContent = text;
    btn.disabled = !enabled;
    if (enabled) btn.addEventListener('click', onClick);
    return btn;
  }

  // Checkout button opens modal
  document.getElementById('checkoutBtn').addEventListener('click', function() {
    if (Object.keys(selectedItems).length === 0) {
      showToast('Please select at least one item', 'warning');
      return;
    }
    renderCheckoutModal();
    openModal('checkoutModal');
  });

  function renderCheckoutModal() {
    const container = document.getElementById('checkoutItems');
    container.innerHTML = '';

    let totalQty = 0;
    let subtotal = 0;

    Object.values(selectedItems).forEach(item => {
      const itemSubtotal = item.price * item.quantity;
      totalQty += item.quantity;
      subtotal += itemSubtotal;

      const row = document.createElement('div');
      row.className = 'checkout-item-row';
      row.innerHTML = `
        <img src="${item.item_icon || '/assets/images/default-item.png'}" class="checkout-item-icon">
        <div class="checkout-item-details">
          <span class="checkout-item-name">${item.item_name}</span>
          <span class="checkout-item-qty">Qty: ${item.quantity} × ₱${item.price.toFixed(2)}</span>
        </div>
        <span class="checkout-item-subtotal">₱${itemSubtotal.toFixed(2)}</span>
        <button class="btn-icon btn-danger btn-sm checkout-item-remove" data-item-id="${item.inventory_id}">
          <span class="material-icons">close</span>
        </button>
      `;
      container.appendChild(row);
    });

    // Remove item from checkout
    container.querySelectorAll('.checkout-item-remove').forEach(btn => {
      btn.addEventListener('click', function() {
        const itemId = this.dataset.itemId;
        delete selectedItems[itemId];
        // Also deselect the card visually
        const card = document.querySelector(`.item-card[data-item-id="${itemId}"]`);
        if (card) card.classList.remove('selected');
        updateCheckoutBar();
        if (Object.keys(selectedItems).length === 0) {
          closeModal('checkoutModal');
          return;
        }
        renderCheckoutModal(); // Re-render
      });
    });

    // Update summary
    document.getElementById('summaryItemCount').textContent = totalQty;
    document.getElementById('summarySubtotal').textContent = '₱' + subtotal.toFixed(2);
    updateTotalWithPayment();
  }

  // Payment type change affects delivery fee display
  document.querySelectorAll('input[name="paymentType"]').forEach(radio => {
    radio.addEventListener('change', function() {
      updateTotalWithPayment();
      // Show/hide address section based on payment type
      const addressSection = document.getElementById('addressSelect').closest('.checkout-section');
      const deliveryFeeRow = document.getElementById('deliveryFeeRow');
      if (this.value === 'pickup') {
        addressSection.style.display = 'none';
        deliveryFeeRow.style.display = 'none';
      } else {
        addressSection.style.display = 'block';
        deliveryFeeRow.style.display = 'flex';
      }
    });
  });

  function updateTotalWithPayment() {
    const paymentType = document.querySelector('input[name="paymentType"]:checked').value;
    const subtotalText = document.getElementById('summarySubtotal').textContent;
    const subtotal = parseFloat(subtotalText.replace('₱', '').replace(',', ''));
    const fee = paymentType === 'pickup' ? 0 : deliveryFee;
    document.getElementById('summaryDeliveryFee').textContent = '₱' + fee.toFixed(2);
    document.getElementById('summaryTotal').textContent = '₱' + (subtotal + fee).toFixed(2);
  }

  // Address dropdown: show custom input when "custom" selected
  document.getElementById('addressSelect').addEventListener('change', function() {
    const customWrap = document.getElementById('customAddressWrap');
    customWrap.style.display = this.value === 'custom' ? 'block' : 'none';
  });

  // Place Order button
  document.getElementById('placeOrderBtn').addEventListener('click', function() {
    const items = Object.values(selectedItems).map(item => ({
      inventory_id: item.inventory_id,
      quantity: item.quantity
    }));

    const paymentType = document.querySelector('input[name="paymentType"]:checked').value;

    // Determine delivery address
    let deliveryAddress = null;
    if (paymentType !== 'pickup') {
      const addressSelect = document.getElementById('addressSelect');
      if (addressSelect.value === 'custom') {
        deliveryAddress = document.getElementById('customAddress').value.trim();
        if (!deliveryAddress) {
          showToast('Please enter a delivery address', 'warning');
          return;
        }
      } else {
        // Send the address_id — server will look up the full address
        deliveryAddress = addressSelect.value; // This is the address_id
      }
    }

    const orderNotes = document.getElementById('orderNotes').value.trim();

    // Disable button, show loading
    this.disabled = true;
    this.innerHTML = '<div class="spinner" style="width:20px;height:20px;"></div> Placing Order...';

    // First: real-time stock validation
    const itemIds = items.map(i => i.inventory_id).join(',');
    ajaxGet('/api/inventory/check_stock.php', { item_ids: itemIds })
      .then(stockData => {
        // Check for out-of-stock items
        const unavailable = [];
        stockData.items.forEach(stockItem => {
          const selected = selectedItems[stockItem.id];
          if (!selected) return;
          if (stockItem.status !== 'active' || stockItem.stock < selected.quantity) {
            unavailable.push(selected.item_name);
            delete selectedItems[stockItem.id];
          }
        });

        if (unavailable.length > 0) {
          Swal.fire({
            icon: 'warning',
            title: 'Items Unavailable',
            html: `The following items are no longer available and have been removed from your order:<br><br><strong>${unavailable.join('<br>')}</strong>`,
            confirmButtonColor: 'var(--primary)'
          });
          renderCheckoutModal();
          updateCheckoutBar();
          resetPlaceOrderBtn();
          if (Object.keys(selectedItems).length === 0) {
            closeModal('checkoutModal');
          }
          return;
        }

        // All items valid — place the order
        return ajaxPost('/api/orders/place_order.php', {
          items: JSON.stringify(items),
          payment_type: paymentType,
          delivery_address: deliveryAddress,
          order_notes: orderNotes
        });
      })
      .then(data => {
        if (!data) return; // Stock validation failed, already handled
        if (data.success) {
          closeModal('checkoutModal');
          Swal.fire({
            icon: 'success',
            title: 'Order Placed Successfully!',
            html: `Your order <strong>#${data.order_id}</strong> has been sent to staff for confirmation.`,
            showCancelButton: true,
            confirmButtonText: 'View Receipt',
            cancelButtonText: 'Go to My Orders',
            confirmButtonColor: 'var(--primary)'
          }).then(result => {
            if (result.isConfirmed) {
              window.open('/receipt.php?token=' + data.receipt_token, '_blank');
              window.location.href = 'my_orders.php';
            } else {
              window.location.href = 'my_orders.php';
            }
          });
        } else {
          showToast(data.message || 'Failed to place order', 'error');
          resetPlaceOrderBtn();
        }
      })
      .catch(err => {
        showToast('An error occurred. Please try again.', 'error');
        resetPlaceOrderBtn();
      });
  });

  function resetPlaceOrderBtn() {
    const btn = document.getElementById('placeOrderBtn');
    btn.disabled = false;
    btn.innerHTML = '<span class="material-icons">check</span> Place Order';
  }

  function preselectReorderItems(reorderItems) {
    reorderItems.forEach(item => {
      const card = document.querySelector(`.item-card[data-item-id="${item.inventory_id}"]`);
      if (card && !card.classList.contains('out-of-stock')) {
        const qtyInput = card.querySelector('.qty-input');
        qtyInput.value = Math.min(item.quantity, parseInt(qtyInput.max));
        selectedItems[item.inventory_id] = {
          inventory_id: item.inventory_id,
          item_name: item.item_name,
          item_icon: item.item_icon,
          price: item.price,
          stock_count: parseInt(qtyInput.max),
          quantity: parseInt(qtyInput.value)
        };
        card.classList.add('selected');
      }
    });
    updateCheckoutBar();
  }

  function showLoading() {
    document.getElementById('itemsLoading').style.display = 'flex';
  }
  function hideLoading() {
    document.getElementById('itemsLoading').style.display = 'none';
  }

});
```

```javascript
document.addEventListener('DOMContentLoaded', function() {

  let currentFilter = 'all';
  let currentPage = 1;
  const perPage = 50; // ITEMS_PER_PAGE

  // Load orders on page load
  loadOrders();

  // Filter buttons
  document.querySelectorAll('#orderFilters .filter-btn').forEach(btn => {
    btn.addEventListener('click', function() {
      document.querySelectorAll('#orderFilters .filter-btn').forEach(b => b.classList.remove('active'));
      this.classList.add('active');
      currentFilter = this.dataset.filter;
      currentPage = 1;
      loadOrders();
    });
  });

  function loadOrders() {
    const params = {
      status: currentFilter === 'all' ? '' : currentFilter,
      page: currentPage
    };

    // Show loading
    document.getElementById('ordersBody').innerHTML =
      '<tr class="loading-row"><td colspan="11"><div class="spinner"></div></td></tr>';
    document.getElementById('emptyOrders').style.display = 'none';

    ajaxGet('/api/orders/get_orders.php', params)
      .then(data => {
        if (data.orders.length === 0) {
          document.getElementById('ordersBody').innerHTML = '';
          document.getElementById('emptyOrders').style.display = 'flex';
          document.getElementById('ordersPagination').innerHTML = '';
          return;
        }

        document.getElementById('emptyOrders').style.display = 'none';
        renderOrders(data.orders);
        renderOrdersPagination(data.total, data.page, data.per_page);
      })
      .catch(err => {
        showToast('Failed to load orders', 'error');
      });
  }

  function renderOrders(orders) {
    const tbody = document.getElementById('ordersBody');
    tbody.innerHTML = '';

    orders.forEach(order => {
      const tr = document.createElement('tr');

      // Determine which actions are available
      const canCancel = (order.status === 'pending'); // Customer can only cancel pending
      const canAccept = (order.status === 'delivered' || order.status === 'picked_up')
                        && !order.customer_confirmed;

      // Items display: combine all order_items into one cell
      const itemsHtml = order.items.map(item => `
        <span class="item-with-icon">
          <img src="${item.item_icon || '/assets/images/default-item.png'}" class="item-icon-sm">
          ${item.item_name}
        </span>
      `).join(', ');

      // Total quantity across all items in this order
      const totalQty = order.items.reduce((sum, item) => sum + parseInt(item.quantity), 0);

      // Payment label
      const paymentLabels = { cod: 'Cash on Delivery', pickup: 'Pickup at Store', online: 'Online' };

      tr.innerHTML = `
        <td>
          ${canCancel ? `<input type="checkbox" class="custom-checkbox order-checkbox" data-order-id="${order.id}">` : ''}
        </td>
        <td><strong>#${order.id}</strong></td>
        <td>${itemsHtml}</td>
        <td>${paymentLabels[order.payment_type] || order.payment_type}</td>
        <td><span class="badge badge-${order.status.replace(/_/g, '-')}">${formatStatus(order.status)}</span></td>
        <td>${totalQty}</td>
        <td>${order.rider_name || '<span class="text-muted">Unassigned</span>'}</td>
        <td>${formatDate(order.order_date)}</td>
        <td>${order.expected_delivery_date ? formatDate(order.expected_delivery_date) : '<span class="text-muted">—</span>'}</td>
        <td>${order.staff_comment || '<span class="text-muted">—</span>'}</td>
        <td class="actions-cell">
          ${canAccept ? `
            <button class="btn-icon btn-success" title="Mark as Received" onclick="acceptOrder(${order.id})">
              <span class="material-icons">check_circle</span>
            </button>
          ` : ''}
          ${canCancel ? `
            <button class="btn-icon btn-danger" title="Cancel Order" onclick="cancelOrder(${order.id})">
              <span class="material-icons">cancel</span>
            </button>
          ` : ''}
          ${order.status === 'cancelled' || order.status === 'delivered' || order.status === 'accepted' || order.status === 'picked_up' ? `
            <button class="btn-icon btn-info" title="Reorder" onclick="reorder(${order.id})">
              <span class="material-icons">replay</span>
            </button>
          ` : ''}
          <button class="btn-icon btn-outline" title="View Receipt" onclick="viewReceipt('${order.receipt_token}')">
            <span class="material-icons">receipt</span>
          </button>
        </td>
      `;

      tbody.appendChild(tr);
    });

    // Init bulk select
    initBulkSelect('ordersTable');
  }

  // Cancel Order
  window.cancelOrder = function(orderId) {
    Swal.fire({
      title: 'Cancel Order #' + orderId + '?',
      html: `
        <p style="margin-bottom: 12px; color: var(--text-secondary);">
          This action cannot be undone. Please provide a reason for cancellation.
        </p>
        <textarea id="cancelReason" class="swal2-textarea"
          placeholder="Reason for cancellation (required)" rows="3"></textarea>
      `,
      icon: 'warning',
      showCancelButton: true,
      confirmButtonText: 'Yes, Cancel Order',
      cancelButtonText: 'No, Keep It',
      confirmButtonColor: '#EF5350',
      preConfirm: () => {
        const reason = document.getElementById('cancelReason').value.trim();
        if (!reason) {
          Swal.showValidationMessage('Please provide a cancellation reason');
          return false;
        }
        return reason;
      }
    }).then(result => {
      if (result.isConfirmed) {
        ajaxPost('/api/orders/cancel_order.php', {
          order_id: orderId,
          reason: result.value
        }).then(data => {
          if (data.success) {
            showToast('Order cancelled successfully', 'success');
            loadOrders(); // Refresh table
            // Check if account was flagged
            if (data.account_flagged) {
              Swal.fire({
                icon: 'warning',
                title: 'Account Flagged',
                text: 'You have reached the maximum number of cancellations. Your ordering privileges have been suspended. You can submit an appeal from the Dashboard.',
                confirmButtonColor: 'var(--warning)'
              });
            }
          } else {
            showToast(data.message || 'Failed to cancel order', 'error');
          }
        });
      }
    });
  };

  // Accept/Received
  window.acceptOrder = function(orderId) {
    Swal.fire({
      title: 'Confirm Receipt',
      text: 'Mark this order as received/accepted? This is optional and just confirms you got your items.',
      icon: 'question',
      showCancelButton: true,
      confirmButtonText: 'Yes, I Received It',
      cancelButtonText: 'Not Yet',
      confirmButtonColor: 'var(--success)'
    }).then(result => {
      if (result.isConfirmed) {
        ajaxPost('/api/orders/confirm_received.php', { order_id: orderId })
          .then(data => {
            if (data.success) {
              showToast('Order marked as received!', 'success');
              loadOrders();
            } else {
              showToast(data.message || 'Failed to confirm receipt', 'error');
            }
          });
      }
    });
  };

  // Reorder
  window.reorder = function(orderId) {
    Swal.fire({
      title: 'Reorder',
      text: 'Would you like to place a new order with the same items?',
      icon: 'question',
      showCancelButton: true,
      confirmButtonText: 'Yes, Reorder',
      confirmButtonColor: 'var(--primary)'
    }).then(result => {
      if (result.isConfirmed) {
        window.location.href = 'place_order.php?reorder=' + orderId;
      }
    });
  };

  // View Receipt
  window.viewReceipt = function(token) {
    window.open('/receipt.php?token=' + token, '_blank');
  };

  // Bulk cancel
  document.getElementById('bulkCancelBtn').addEventListener('click', function() {
    const selectedIds = getSelectedIds('ordersTable');
    if (selectedIds.length === 0) {
      showToast('No orders selected', 'warning');
      return;
    }

    Swal.fire({
      title: `Cancel ${selectedIds.length} Order(s)?`,
      html: `<textarea id="bulkCancelReason" class="swal2-textarea"
              placeholder="Reason for cancellation (required)" rows="3"></textarea>`,
      icon: 'warning',
      showCancelButton: true,
      confirmButtonText: 'Cancel All Selected',
      confirmButtonColor: '#EF5350',
      preConfirm: () => {
        const reason = document.getElementById('bulkCancelReason').value.trim();
        if (!reason) {
          Swal.showValidationMessage('Please provide a reason');
          return false;
        }
        return reason;
      }
    }).then(result => {
      if (result.isConfirmed) {
        ajaxPost('/api/orders/bulk_update.php', {
          order_ids: JSON.stringify(selectedIds),
          action: 'cancel',
          reason: result.value
        }).then(data => {
          if (data.success !== undefined) {
            showToast(`${data.success} order(s) cancelled`, 'success');
            loadOrders();
          }
        });
      }
    });
  });

  // Select all checkbox handler
  document.getElementById('selectAllOrders').addEventListener('change', function() {
    const checkboxes = document.querySelectorAll('.order-checkbox');
    checkboxes.forEach(cb => { cb.checked = this.checked; });
    updateBulkBar();
  });

  // Delegate individual checkbox changes
  document.getElementById('ordersBody').addEventListener('change', function(e) {
    if (e.target.classList.contains('order-checkbox')) {
      updateBulkBar();
    }
  });

  function updateBulkBar() {
    const checked = document.querySelectorAll('.order-checkbox:checked').length;
    const bar = document.getElementById('bulkActions');
    bar.style.display = checked > 0 ? 'flex' : 'none';
    document.getElementById('bulkSelectedCount').textContent = checked;
  }

  // Helper functions
  function formatStatus(status) {
    const labels = {
      pending: 'Pending',
      confirmed: 'Confirmed',
      assigned: 'Assigned',
      on_delivery: 'On Delivery',
      delivered: 'Delivered',
      accepted: 'Accepted',
      ready_for_pickup: 'Ready for Pickup',
      picked_up: 'Picked Up',
      cancelled: 'Cancelled'
    };
    return labels[status] || status;
  }

  function formatDate(dateStr) {
    if (!dateStr) return '—';
    const d = new Date(dateStr);
    return d.toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' })
      + ' ' + d.toLocaleTimeString('en-US', { hour: '2-digit', minute: '2-digit' });
  }

  function renderOrdersPagination(total, page, perPage) {
    // Same pagination rendering logic as place_order.js
    const totalPages = Math.ceil(total / perPage);
    const container = document.getElementById('ordersPagination');
    container.innerHTML = '';
    if (totalPages <= 1) return;

    const prevBtn = document.createElement('button');
    prevBtn.className = 'page-btn';
    prevBtn.textContent = 'Previous';
    prevBtn.disabled = page <= 1;
    prevBtn.addEventListener('click', () => { currentPage--; loadOrders(); });
    container.appendChild(prevBtn);

    for (let i = Math.max(1, page-2); i <= Math.min(totalPages, page+2); i++) {
      const pb = document.createElement('button');
      pb.className = `page-btn ${i === page ? 'active' : ''}`;
      pb.textContent = i;
      pb.addEventListener('click', () => { currentPage = i; loadOrders(); });
      container.appendChild(pb);
    }

    const nextBtn = document.createElement('button');
    nextBtn.className = 'page-btn';
    nextBtn.textContent = 'Next';
    nextBtn.disabled = page >= totalPages;
    nextBtn.addEventListener('click', () => { currentPage++; loadOrders(); });
    container.appendChild(nextBtn);

    const info = document.createElement('span');
    info.className = 'pagination-info';
    info.textContent = `Showing ${(page-1)*perPage + 1}–${Math.min(page*perPage, total)} of ${total} orders`;
    container.appendChild(info);
  }

});
```

```javascript
document.addEventListener('DOMContentLoaded', function() {

  let currentFilter = 'all';
  let currentPage = 1;
  let searchText = '';
  let dateFrom = '';
  let dateTo = '';

  loadHistory();

  // Filter buttons
  document.querySelectorAll('#historyFilters .filter-btn').forEach(btn => {
    btn.addEventListener('click', function() {
      document.querySelectorAll('#historyFilters .filter-btn').forEach(b => b.classList.remove('active'));
      this.classList.add('active');
      currentFilter = this.dataset.filter;
      currentPage = 1;
      loadHistory();
    });
  });

  // Search (debounced)
  let searchTimeout;
  document.getElementById('historySearch').addEventListener('input', function() {
    clearTimeout(searchTimeout);
    searchTimeout = setTimeout(() => {
      searchText = this.value.trim();
      currentPage = 1;
      loadHistory();
    }, 300);
  });

  // Date range
  document.getElementById('dateFrom').addEventListener('change', function() {
    dateFrom = this.value;
    currentPage = 1;
    loadHistory();
  });
  document.getElementById('dateTo').addEventListener('change', function() {
    dateTo = this.value;
    currentPage = 1;
    loadHistory();
  });

  function loadHistory() {
    const params = {
      status: currentFilter === 'all' ? '' : currentFilter,
      search: searchText,
      page: currentPage,
      date_from: dateFrom,
      date_to: dateTo,
      scope: 'history' // Tell the API to include completed/cancelled orders
    };

    document.getElementById('historyBody').innerHTML =
      '<tr class="loading-row"><td colspan="13"><div class="spinner"></div></td></tr>';
    document.getElementById('emptyHistory').style.display = 'none';

    ajaxGet('/api/orders/get_orders.php', params)
      .then(data => {
        if (data.orders.length === 0) {
          document.getElementById('historyBody').innerHTML = '';
          document.getElementById('emptyHistory').style.display = 'flex';
          document.getElementById('historyPagination').innerHTML = '';
          return;
        }

        document.getElementById('emptyHistory').style.display = 'none';
        renderHistory(data.orders);
        renderPagination(data.total, data.page, data.per_page, 'historyPagination', loadHistory);
      })
      .catch(() => showToast('Failed to load order history', 'error'));
  }

  function renderHistory(orders) {
    const tbody = document.getElementById('historyBody');
    tbody.innerHTML = '';

    orders.forEach(order => {
      const itemsHtml = order.items.map(item =>
        `<span class="item-with-icon"><img src="${item.item_icon || '/assets/images/default-item.png'}" class="item-icon-sm">${item.item_name}</span>`
      ).join(', ');

      const totalQty = order.items.reduce((sum, i) => sum + parseInt(i.quantity), 0);
      const paymentLabels = { cod: 'Cash on Delivery', pickup: 'Pickup at Store', online: 'Online' };
      const deliveryLabels = { delivery: 'Delivery', pickup: 'Pickup' };

      const tr = document.createElement('tr');
      tr.innerHTML = `
        <td><strong>#${order.id}</strong></td>
        <td>${itemsHtml}</td>
        <td>${paymentLabels[order.payment_type] || order.payment_type}</td>
        <td>${deliveryLabels[order.delivery_type] || order.delivery_type}</td>
        <td><span class="badge badge-${order.status.replace(/_/g, '-')}">${formatStatus(order.status)}</span></td>
        <td>${totalQty}</td>
        <td><strong>${formatCurrency(order.total_amount)}</strong></td>
        <td>${order.rider_name || '<span class="text-muted">N/A</span>'}</td>
        <td>${formatDate(order.order_date)}</td>
        <td>${order.delivered_at ? formatDate(order.delivered_at) : '<span class="text-muted">—</span>'}</td>
        <td class="truncate-cell" title="${order.cancellation_reason || ''}">${order.cancellation_reason || '<span class="text-muted">—</span>'}</td>
        <td>${order.cancelled_by_name ? order.cancelled_by_name : '<span class="text-muted">—</span>'}</td>
        <td class="actions-cell">
          <button class="btn-icon btn-info" title="Reorder" onclick="reorder(${order.id})">
            <span class="material-icons">replay</span>
          </button>
          <button class="btn-icon btn-outline" title="View Receipt" onclick="viewReceipt('${order.receipt_token}')">
            <span class="material-icons">receipt</span>
          </button>
        </td>
      `;
      tbody.appendChild(tr);
    });
  }

  // Reorder and receipt functions (same as my_orders.js)
  window.reorder = function(orderId) {
    Swal.fire({
      title: 'Reorder',
      text: 'Place a new order with the same items?',
      icon: 'question',
      showCancelButton: true,
      confirmButtonText: 'Yes, Reorder',
      confirmButtonColor: 'var(--primary)'
    }).then(result => {
      if (result.isConfirmed) {
        window.location.href = 'place_order.php?reorder=' + orderId;
      }
    });
  };

  window.viewReceipt = function(token) {
    window.open('/receipt.php?token=' + token, '_blank');
  };

  // Shared helpers
  function formatStatus(s) {
    const labels = { pending:'Pending', confirmed:'Confirmed', assigned:'Assigned', on_delivery:'On Delivery', delivered:'Delivered', accepted:'Accepted', ready_for_pickup:'Ready for Pickup', picked_up:'Picked Up', cancelled:'Cancelled' };
    return labels[s] || s;
  }
  function formatDate(d) {
    if (!d) return '—';
    return new Date(d).toLocaleDateString('en-US', { month:'short', day:'numeric', year:'numeric' }) + ' ' + new Date(d).toLocaleTimeString('en-US', { hour:'2-digit', minute:'2-digit' });
  }
  function formatCurrency(a) { return '₱' + parseFloat(a).toFixed(2); }

  function renderPagination(total, page, perPage, containerId, loadFunc) {
    const totalPages = Math.ceil(total / perPage);
    const container = document.getElementById(containerId);
    container.innerHTML = '';
    if (totalPages <= 1) return;

    const prev = document.createElement('button');
    prev.className = 'page-btn'; prev.textContent = 'Previous'; prev.disabled = page <= 1;
    prev.addEventListener('click', () => { currentPage--; loadFunc(); });
    container.appendChild(prev);

    for (let i = Math.max(1,page-2); i <= Math.min(totalPages,page+2); i++) {
      const pb = document.createElement('button');
      pb.className = `page-btn ${i===page?'active':''}`;
      pb.textContent = i;
      pb.addEventListener('click', () => { currentPage = i; loadFunc(); });
      container.appendChild(pb);
    }

    const next = document.createElement('button');
    next.className = 'page-btn'; next.textContent = 'Next'; next.disabled = page >= totalPages;
    next.addEventListener('click', () => { currentPage++; loadFunc(); });
    container.appendChild(next);

    const info = document.createElement('span');
    info.className = 'pagination-info';
    info.textContent = `Showing ${(page-1)*perPage+1}–${Math.min(page*perPage,total)} of ${total}`;
    container.appendChild(info);
  }

});
```

## IMPORTANT IMPLEMENTATION NOTES FOR CUSTOMER PAGES

1. **Account Status Handling on Every Page:**
   - If `$_SESSION['status'] === 'pending'`: Show the "Account Pending Approval" notice. Hide or disable all functional content. Only Settings page should be partially accessible (profile viewing).
   - If `$_SESSION['status'] === 'flagged'`: Show the "Account Flagged" warning on dashboard. Allow viewing orders and history. Block "Place Order" page (redirect to dashboard with toast message). Allow Settings access including appeal submission.
   - If `$_SESSION['status'] === 'active'`: Full access to everything.

2. **Every page must use the shared `ajaxPost()` and `ajaxGet()` helper functions** from `global.js` which automatically attach the CSRF token.

3. **Every table must implement pagination** showing max 50 items per page with Previous/Next buttons and a "Showing X-Y of Z items" counter.

4. **All date/time values** displayed to the user must be formatted in Philippines Manila timezone using the helper functions.

5. **Item icons** in tables and cards: If the item has an `item_icon` path, show that image. If NULL/empty, show `/assets/images/default-item.png`.

6. **Status badges** must use the corresponding CSS class: `badge-pending`, `badge-confirmed`, etc. Use the `get_status_badge_class()` PHP function server-side or the JS `formatStatus()` function client-side.

7. **The "Reorder" feature** works by redirecting to `place_order.php?reorder={order_id}`. On that page, PHP fetches the original order's items via `api/orders/reorder.php` and passes them to JS as a JSON variable. JS then pre-selects those items in the grid after items load.

8. **The "View Receipt" button** opens `receipt.php?token={receipt_token}` in a new tab. This is a public page that was built in Prompt #1.

9. **Dark mode** is already handled by the global CSS variables and the `data-theme` attribute on `<html>`. No additional dark mode code is needed in page-specific CSS — just use CSS variables (`var(--text-primary)`, `var(--surface)`, etc.) for all colors.

10. **The sidebar active state** is handled by `sidebar.js` which matches the current URL path to sidebar link hrefs. Each sidebar link should point to the correct relative path (e.g., `dashboard.php`, `place_order.php`, etc.).

11. **Floating label animation** for inputs: The `float-input-group` component has CSS that moves the label up when the input is focused or has a value. Use `placeholder=" "` (a space) on inputs to trigger the `:not(:placeholder-shown)` CSS selector for the filled state.

12. **Glass card effect** should be used sparingly — primarily for main content cards like settings sections, stat cards, and the dashboard pending/flagged notices. Not on every element.

13. **All customer-facing tables** should show human-readable values: "Cash on Delivery" instead of "cod", "On Delivery" instead of "on_delivery", etc. Use mapping objects in JS.

14. **Each page-specific JS file** must be wrapped in `document.addEventListener('DOMContentLoaded', function() { ... })` to ensure DOM is ready.

15. **Error handling**: Every AJAX call must have `.catch()` that shows a toast with a generic error message. Never leave failed requests silently unhandled.
